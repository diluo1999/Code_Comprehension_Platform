Python naming conventions discourage long variable and function names. This function's name is unnecessarily long; we know it will add two numbers by the nature of its parameters. We can easily reduce it. 
def thisAddsTwoNumbers(number1, number2):
    return number1 + number2
Instead, consider the following: 
def add(n1, n2):
    return n1 + n2
